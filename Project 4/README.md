# ET721 Lab - Student Learning Management App

A Django-based web application for students to manage tasks, write blog posts about their learning journey, and upload/organize notes as images.

## Features
- **To-Do List**: Create, edit, delete, mark complete tasks with categories and due dates.
- **Blog**: Write and publish posts, view personal posts, add/remove likes.
- **Notes Upload**: Upload images of notes, categorize by subject, preview and download.
- User authentication (login/logout)
- All features are user-specific (private data)

## Tech Stack
- Backend: Django
- Database: SQLite (development)
- Frontend: HTML, CSS (no JS frameworks)
